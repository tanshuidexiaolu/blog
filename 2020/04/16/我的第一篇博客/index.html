我的第一篇博客

坎

我这人吧，很怪。不是我很怪，是我无论学什么都不能一次到位。

Java

当初我在学Java的时候，第一个代码：HelloWorld。我那是怎么也出不来，我就很着急，但着急有什么用呢。我这个人，爱钻牛角尖，我当时运行不了，死找！找了一下午，却发现我没改后缀名…，————当时我的电脑没有设置显示后缀名所以我的代码文件不过是HelloWorld.java.txt罢了，后来干完后又报错，这个倒好办，是我忘记写分号了，众所周知，分号是程序员最容易忘记写的符号之一（除Python程序员外）。

EXE4j

java还没学完，光学了个皮毛（数据类型，程序流程），就去想打包exe了，看网上的教程好像可以用一个叫EXE4j的软件，看起来很简单，但是我亲自搞的时候却报错了。我为了让其他电脑也可以，我决定把系统环境变量绑定jre改成携带jre，当时怎么做的差不多忘了，不过好像当时我就根本没有放jre，因为我是后来才知道jdk1.8自带jre的，可能当时我连jre是啥都忘了。后来试了试，好了。因为也许我学了面向对象什么的

未完待续（最近要返校有点忙）
